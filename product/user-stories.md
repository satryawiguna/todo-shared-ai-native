# User Stories

## Persona

- **Andi (Pengguna Umum)**: Karyawan kantoran yang ingin melacak tugas harian. Menggunakan aplikasi dari laptop saat bekerja.

---

## Epic 1: Manajemen Todo Dasar

### US-01: Melihat Daftar Todo
> Sebagai **Andi**, saya ingin **melihat daftar semua todo saya** sehingga **saya tahu apa yang harus dikerjakan**.

**Story Points:** 3

### US-02: Membuat Todo Baru
> Sebagai **Andi**, saya ingin **membuat todo baru dengan judul dan prioritas** sehingga **saya dapat mencatat tugas yang harus dilakukan**.

**Story Points:** 2

### US-03: Melihat Detail Todo
> Sebagai **Andi**, saya ingin **melihat detail satu todo** sehingga **saya dapat membaca deskripsi lengkapnya**.

**Story Points:** 1

### US-04: Mengubah Todo
> Sebagai **Andi**, saya ingin **mengubah judul, deskripsi, status, atau prioritas todo** sehingga **informasi todo tetap akurat**.

**Story Points:** 2

### US-05: Menghapus Todo
> Sebagai **Andi**, saya ingin **menghapus todo yang sudah tidak relevan** sehingga **daftar todo saya tetap bersih**.

**Story Points:** 1

---

## Epic 2: Organisasi Todo

### US-06: Filter Berdasarkan Status
> Sebagai **Andi**, saya ingin **memfilter todo berdasarkan status (aktif/selesai)** sehingga **saya fokus pada tugas yang belum selesai**.

**Story Points:** 2

### US-07: Filter Berdasarkan Prioritas
> Sebagai **Andi**, saya ingin **memfilter todo berdasarkan prioritas (rendah/sedang/tinggi)** sehingga **saya dapat mengerjakan yang paling penting terlebih dahulu**.

**Story Points:** 2

### US-08: Mencari Todo
> Sebagai **Andi**, saya ingin **mencari todo berdasarkan kata kunci di judul** sehingga **saya cepat menemukan todo tertentu**.

**Story Points:** 1

### US-09: Menandai Todo Selesai
> Sebagai **Andi**, saya ingin **menandai todo sebagai selesai** sehingga **saya bisa melacak progres saya**.

**Story Points:** 1
