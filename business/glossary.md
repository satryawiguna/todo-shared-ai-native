# Glossary

| Term | Definisi |
|------|----------|
| Task | Unit pekerjaan yang dapat dilacak oleh user |
| Status | State lifecycle sebuah task: todo, in_progress, done |
| Priority | Tingkat urgensi task: low, medium, high |
| Due Date | Tanggal target penyelesaian task (opsional) |
| JWT | JSON Web Token, digunakan untuk autentikasi stateless |
| Soft Delete | Penghapusan logis menggunakan field `deleted_at`, data tidak dihapus dari DB |
| Owner | User yang membuat sebuah task |
