# Roadmap

## MVP (Fase 1) — Sekarang

- [ ] CRUD Todo (buat, baca, ubah, hapus)
- [ ] Daftar todo dengan paginasi
- [ ] Filter berdasarkan status (semua, aktif, selesai)
- [ ] Filter berdasarkan prioritas (rendah, sedang, tinggi)
- [ ] Pencarian berdasarkan judul
- [ ] Validasi input (judul wajib, maksimal 200 karakter)
- [ ] Soft delete (todo dihapus tetap tersimpan 30 hari)
- [ ] Docker support (frontend + backend + database)

## v1 (Fase 2)

- [ ] Autentikasi pengguna (register, login, JWT)
- [ ] Multi-user — setiap pengguna hanya melihat todo miliknya
- [ ] Kategori/tag untuk todo
- [ ] Due date dengan pengingat
- [ ] Dark mode

## v2 (Fase 3)

- [ ] Kolaborasi tim (bagi todo dengan anggota tim)
- [ ] Notifikasi email untuk deadline
- [ ] Dashboard analitik (todo selesai per minggu, dll.)
- [ ] Mobile app (React Native / PWA)

## Prioritas Fitur

| Prioritas | Definisi |
|---|---|
| **P0** | Wajib untuk MVP — tidak bisa diluncurkan tanpa ini |
| **P1** | Penting — harus ada segera setelah MVP |
| **P2** | Nice-to-have — dikerjakan jika ada waktu |
