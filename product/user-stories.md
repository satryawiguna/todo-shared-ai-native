# User Stories

## Authentication
- US-01: Sebagai user baru, saya ingin registrasi dengan email & password agar dapat menggunakan aplikasi
- US-02: Sebagai user, saya ingin login agar dapat mengakses task saya
- US-03: Sebagai user, saya ingin tetap login selama token masih valid tanpa re-login

## Task Management
- US-10: Sebagai user, saya ingin membuat task baru dengan detail lengkap
- US-11: Sebagai user, saya ingin mengedit task agar informasinya tetap akurat
- US-12: Sebagai user, saya ingin menghapus task yang tidak relevan
- US-13: Sebagai user, saya ingin mengubah status task sesuai progress
- US-14: Sebagai user, saya ingin memfilter task agar mudah menemukan yang relevan
- US-15: Sebagai user, saya ingin mencari task berdasarkan kata kunci
